# Тык-кликер

Тык-кликер - небольшая Windows-программа на Python, которая реагирует на левый клик мыши в любом месте экрана: проигрывает звук клика и кратко показывает серую надпись "тык" рядом с курсором.

## Требования

- Python 3.8-3.13
- Windows 10/11

## Установка

```bash
pip install -r requirements.txt
```

`tkinter` и `winreg` входят в стандартную поставку Python для Windows.

Если установка `pygame` завершается ошибкой `Failed to build 'pygame'`, проверьте версию Python:

```bash
python --version
```

Для Python 3.14 может не быть готового wheel-файла `pygame`, из-за чего pip пытается собрать библиотеку из исходников. Установите Python 3.12 или 3.13 с [python.org](https://www.python.org/downloads/windows/) и ставьте зависимости через тот же интерпретатор:

```bash
py -3.12 -m pip install -r requirements.txt
py -3.12 main.py
```

## Настройка звука

Поместите свой файл клика в:

```text
sounds/click.mp3
```

В репозитории лежит только локальная заглушка, а настоящий звук пользователь добавляет сам. Бесплатные звуки можно найти на [freesound.org](https://freesound.org/).

Если `sounds/click.mp3` отсутствует или не читается, программа продолжит работать без звука.

## Запуск

```bash
python main.py
```

## Управление

- Левый клик мыши - эффект "тык".
- Иконка в трее - настройки программы.
- Чекбокс "Запускать при старте Windows" - программа будет запускаться вместе с Windows.
- Крестик окна настроек скрывает окно в трей.
- В меню трея доступны "Показать окно" и "Выход".

## Структура проекта

```text
tykclicker/
├── main.py
├── README.md
├── .gitignore
├── settings.json
└── sounds/
    └── click.mp3
```

`settings.json` создается при первом изменении настроек и не включается в репозиторий. `sounds/click.mp3` также игнорируется Git, чтобы пользователь мог заменить звук своим файлом.
